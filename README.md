# Unsensored Model Endpoints

Custom connector for OpenRouter's unsensored model endpoints.

## Overview

This repository contains OpenAPI/Swagger specifications for connecting to OpenRouter's unsensored language models and chat endpoints.

## Files

- `connectors/OpenRouter/OpenRouter_unsensored-model-endpoints_swagger.yml` - OpenAPI specification for the OpenRouter unsensored model endpoints connector

## Usage

This connector can be imported into platforms that support OpenAPI/Swagger specifications for custom API integrations.
